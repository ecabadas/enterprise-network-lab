# Enterprise Network Configuration & Troubleshooting Lab

## Overview
Virtual enterprise network lab using Windows and Linux environments to build hands-on experience with IP configuration, DNS, DHCP concepts, and network troubleshooting.

## Environment
- VirtualBox
- Windows 10/11 virtual machine
- Ubuntu Linux virtual machine

## Skills Practiced
- Network configuration (IP addressing, subnetting basics)
- DNS and DHCP concepts
- Cross-platform connectivity (Windows ↔ Linux)
- Network troubleshooting using command-line tools
- Traffic analysis using Wireshark

## Tools Used
- ping
- tracert / traceroute
- ipconfig / ifconfig
- Wireshark

## Lab Focus Areas
- Virtual network setup between VMs
- Static and dynamic IP configuration
- Connectivity testing and troubleshooting scenarios
- Basic network security awareness

## Status
Active project – building and documenting lab exercises
